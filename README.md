# react-native-mantou 

馒头 - 互联网抱团成长社群

## 环境 Requirements

* React Native: 0.44

* Xcode: 8.3.2

* webstrom: 2016.2

## 使用 Usage

第一步：下载依赖库
```javascript
npm install
```
第二步：启动运行
```javascript
react-native run-ios
```
